## KageriEditor

**日本語を使うライター・ブロガーのためのテキストエディタです。Mac用とAndroid用を開発中です。**

保存ダイアログは出ません。本文の1行目がそのままファイル名になり、⌘Sまたは保存ボタンでサッと保存できます。折り返しも1行と数えるライター向けの行番号、全角換算の文字数カウント、句読点や閉じ括弧が行頭に来ないよう自動調整しながら改行を挿入する「整形」機能など、原稿を書くために必要な機能を搭載しました。

### 特徴

- **ダイアログレス設計** — 保存も閉じるも確認ダイアログなし。書くことに集中できます
- **全角換算の文字数カウント** — ステータスバーに常時表示。選択中の文字数もひと目で分かります
- **行頭禁則（整形）** — 句読点・閉じ括弧・小書き文字などが行頭に来ないよう、指定した文字数で自動的に折り返します。また、改行を除去する非整形機能では、行頭スペースや記号などの段落冒頭を認識します
- **タブでの複数文書編集（Android版）**、検索結果一覧（ヒット箇所の前後文脈つき）、日付・時刻の挿入など、原稿執筆に便利な機能を搭載
- **タッチ操作向けの独自ジョイスティック（Android版）** — 画面下部のパッドで、キーボードを介さず正確にカーソル移動・選択ができます
- **完全無料・広告なし・アカウント不要** — INTERNET権限自体を持たないため、原稿の内容が外部に送信されることはありません
- **カスタマイズ可能なメニューバー** — よく使う機能だけを好きな順番で並べられます
### プライバシー

本アプリは利用者の情報を一切収集しません。ネットワーク通信の権限を持たないため、書いた文章が外部に送信されることは技術的にありません。詳しくは[プライバシーポリシー](https://github.com/KuriharaKageri/KageriEditor/blob/main/PrivacyPolicy.md) をご覧ください。
